💹 ARTICS-NFTs ⁣
⁣
Desarrollo de WebApp con tecnologías Fullstack:⁣
⁣
Backend: JAVA con Spring - Spring Boot (Spring Security, JPA, Rest, JDBC, Test), Lombock, ItextPDF.⁣
⁣
Solicitudes HTTP manejadas con Axios⁣
Base de Datos SQL H2⁣
⁣
Frontend: Javascript con Vue.Js, HTML5, CSS, Bootstrap, SweetAlert2.⁣
⁣
Principales funcionalidades:⁣
⁣
1) Registro de usuarios (autenticacion y autorizacion).
2) Servicios para la creación de NFTs.⁣
3) Api RESTfull.⁣
4) Carro de compras.⁣
5) Sección de favoritos. ⁣
6) Borrado lógico de elementos para la vista del usuario pero con persistencia de los datos en la base de datos.
⁣7) Edición de NFTs.
8) Pagos con balance de cuenta.⁣
9) Detalle de compra (pdf).
10) Validador de cuenta por email.
11) Validador de emali.
